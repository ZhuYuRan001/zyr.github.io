An open-source personal blog with a pixel-art aesthetic.

## Features

- Astro 7 with Rust-powered performance
- React islands for interactive components
- MDX for content authoring
- Tailwind CSS for styling
- Framer Motion animations
- Dark mode support
- RSS and Sitemap

## Tech Stack

- **Framework**: Astro 7
- **UI**: React 19
- **Styling**: Tailwind CSS 4
- **Content**: MDX with Sätteri
- **Animations**: Framer Motion
- **Fonts**: Press Start 2P, Pixelify Sans
- **Comments**: Giscus
- **Desktop UI**: react-rnd
